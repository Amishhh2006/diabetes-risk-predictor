# Diabetes Risk Predictor

This is a Streamlit web app that predicts a person's diabetes risk based on input health parameters using a machine learning model trained on the Pima Indians Diabetes dataset.

## How to Use

1. Clone this repo.
2. Run `pip install -r requirements.txt`
3. Launch the app with `streamlit run app.py`

## Disclaimer

This app is for demonstration purposes only and not a substitute for professional medical advice.
